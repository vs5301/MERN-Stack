Making HTTP Requests with XMLHttpRequest

Introduction

XMLHttpRequest is a browser-based API that can be used to send HTTP requests and receive HTTP responses. It plays a crucial role in AJAX programming and allows for asynchronous communication between a client and a server.

Creating an XMLHttpRequest Object

The first step in using the XMLHttpRequest API is to create a new instance of the object. 

                
                    const xhr = new XMLHttpRequest()

Initializing a Request

To make an HTTP request, you'll need to initialize it using the open() method 

                
                    xhr.open('GET', 'https://api.example.com/data', true)

Sending the Request

Once the request has been initialized, the send() method is used to send it. 

                
                    xhr.send()

Handling Responses

You can use the onload and oneerror events to handle successful and failed requests, respectively. 

                
                    xhr.onload = function(){
                        if(xhr.status >= 200 && xhr.status < 3){
                            console.log('Success: ' , xhr.responseText)
                        } else{
                            console.log('Failed: ', xhr.status)
                        }
                    }

                    xhr.oneerror = function(){
                        console.log('Request error...')
                    }

Complete Example

                
                    const xhr = new XMLHttpRequest()

                    xhr.open('GET', 'https://api.example.com/data', true)

                    xhr.send()

                    xhr.onload = function(){
                        if(xhr.status >= 200 && xhr.status < 3){ 
                            console.log('Success: ' , xhr.responseText)
                        } else{
                            console.log(' Failed: ', xhr.status)
                        }
                    }
                    
                    xhr.oneerror = function(){
                        console.log(' Request error...') 
                    }

Conclusion

XMLHttpRequest is a powerful tool for making HTTP requests and working with AJAX. While newer technologies like Fetch and Axios are now available, understanding XMLHttpRequest is essential for grasping the underlying mechanics of client-server communication.



Making HTTP Requests with the Fetch API

Introduction

The Fetch API provides a modern approach to make HTTP requests and handle responses. It returns Promises and is much cleaner and more powerful than the older XMLHttpRequest.

Basic GET Request

To perform a GET request, you can use the fetch() function: 

                
                    fetch('https://api.example.com/data')
                        .then(response => response.json())
                        .then(data => console.log(data))
                        .catch(error => console.log('Error: ', error))

Using async/await

You can also use the Fetch API with async/await for better readabilty. 

                
                    async function fetchData(){
                        try{
                            const response = await fetch('https://api.example.com/data')
                            const data = await response.json()
                            console.log(data)
                        } catch(error) {
                            console.log('Error: ', error)
                        }
                    }

                    fetchData()

POST Request

To make a POST request, you need to pass an options object to the fetch() function: 

                
                    fetch('https://api.example.com/data', {
                        method: 'POST',
                        headers: {
                            'Content-Type' : 'application/json'
                        },
                        body: JSON.stringify({key: 'value'})
                    })
                    .then(response => response.json())
                    .then(data => console.log(data))
                    .catch(error => console.log('Error: ', error))

Conclusion

Understanding how to use the Fetch API for making HTTP requests will help you manage client-server interactions in a more modern and cleaner way. It is more flexible and powerful than its predecessor, XMLHttpRequest, and is widely used in moder web development.



Making HTTP Requests with Axios

Introduction

Axios is a promise-based HTTP client for JavaScript that can be used in both the browser and Node.js environments. It has a straightforward API and can handle requests and responses in JSON format, making it ideal for use with modern web applications.

Installing Axios

You can include Axios in your product by using npm or by including a CDN link in your HTML file. 

                
                    // Using npm
                    npm install axios

                    // Using CDN

Basic GET Request

A basic GET request using Axios is simple to implement: 

                
                    axios.get('https://api.example.com/items')
                        .then(response => {
                            console.log(response.data)
                        }).catch(error => {
                            console.log('Error fetching data: ', error)
                        })

Using async/await

Axios works seemlessly with async/await: 

                
                    async function fetchData(){
                        try{
                            const response = await axios.get('https://api.example.com/items')
                            console.log(response.data)
                        } catch(error){
                            console.log('Error fetching data: ', error)
                        }
                    }

                    fetchData()

POST Request

To perform a POST request, you can do the following: 

                
                    axios.post('https://api.example.com/items', {
                        name: 'New Item',
                        description: 'New Item Description'
                    }).then(response => {
                        console.log('Data has been send: ', response)
                    }).catch(error => {
                        console.log('Error sending data: ', error)
                    })

Conclusion

Axios provides a clean and easy-to-undertand API for making HTTP requests. It also integrates well with modern JavaScript features like promises and async/await, making it a choice for your HTTP needs.
